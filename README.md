# 🎧 Decibelx — Audio E-commerce Web Application

Welcome to **Decibelx**, a full-stack e-commerce platform focused on audio products like earbuds, headphones, and neckbands. This project is built with **Node.js**, **Express.js**, **MongoDB Atlas**, and **EJS**, and features powerful user/admin functionalities, secure authentication, and real payment integration.

🌐 **Live Website**: [decibelx.shop](https://decibelx.shop)

---

## 🚀 Features

### 👥 User Side

- 🔐 **Google Authentication & Email OTP Signup/Login**
- 📧 **Email OTP verification** for sensitive actions (signup, update email/password, etc.)
- 🎁 **Referral system** with user-specific referral codes
- 🛍️ **Product browsing** with modern UI (Home, Shop, About, Contact)
- 💳 **Payment Options**:
  - Razorpay (online payment)
  - Wallet
  - Cash on Delivery (COD)
- 📝 **User Profile Management**:
  - Update name, email, mobile, password
  - Manage multiple addresses
- 📦 **Order Management**:
  - View orders and status
  - Cancel and return requests

---

### 🛠️ Admin Side

- 📊 **Dashboard**:
  - Sales analytics with charts and graphs
  - Revenue and order calculations
- 📦 **Product Management**:
  - Add, edit, delete products
- 🎟️ **Coupon & Offer Management**
- 👥 **User Management**:
  - Block/unblock users
- 📬 **Order Management**:
  - View and update order status (placed, shipped, delivered, etc.)

---

## 🛠️ Tech Stack

| Frontend   | Backend         | Database      | Other Integrations |
|------------|------------------|---------------|---------------------|
| EJS        | Node.js          | MongoDB Atlas | Razorpay API        |
| Bootstrap  | Express.js       | Mongoose ORM  | Google Auth, OTP    |
